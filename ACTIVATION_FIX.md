# 许可证激活问题修复说明

## 问题描述

用户输入认证密钥 `5B9A-212C-54D7-ADE5` 后点击激活，显示错误 "license_key not found"。

## 问题原因

许可证系统仍在尝试调用 LemonSqueezy API 进行在线验证，但我们使用的是闲鱼（离线销售）模式，不需要在线验证。

## 修复内容

### 1. 更新 `activate()` 函数
- ✅ 改为本地验证密钥格式
- ✅ 移除 LemonSqueezy API 调用
- ✅ 支持大小写不敏感（自动转为大写）
- ✅ 验证格式：`XXXX-XXXX-XXXX-XXXX`（4组4位十六进制）

### 2. 更新 `validate()` 函数
- ✅ 改为本地验证
- ✅ 移除在线 API 调用
- ✅ 只检查密钥格式是否正确

### 3. 更新 `deactivate()` 函数
- ✅ 改为本地清除
- ✅ 移除在线 API 调用

### 4. 更新 `isCacheValid()` 函数
- ✅ 移除 7 天缓存过期检查
- ✅ 本地激活后永久有效

### 5. 更新 `init()` 函数
- ✅ 移除启动时的在线验证
- ✅ 只加载本地许可证数据

### 6. 更新 `startPeriodicCheck()` 函数
- ✅ 禁用定期在线验证
- ✅ 本地激活后不需要定期检查

## 使用方法

### 1. 生成许可证密钥

```bash
npm run gen
```

密钥保存在 `licenses.txt` 文件中。

### 2. 激活许可证

1. 启动应用：`npm start`
2. 进入"设置" Tab
3. 在"许可证激活"区域输入密钥
4. 点击"激活"按钮
5. 看到"激活成功！"提示

### 3. 验证激活

激活成功后：
- 顶部横幅显示"Pro 已激活"
- Pro 功能解锁（拍照监控、数学题门禁、详细统计等）
- 许可证信息显示在设置页面

## 密钥格式

### 正确格式
- ✅ `5B9A-212C-54D7-ADE5`（标准格式）
- ✅ `5b9a-212c-54d7-ade5`（小写，会自动转为大写）
- ✅ `A3F2-B8D1-C4E9-F7A6`（其他有效密钥）

### 错误格式
- ❌ `5B9A212C54D7ADE5`（缺少短横线）
- ❌ `5B9A-212C-54D7`（不完整）
- ❌ `5B9A-212C-54D7-XXXX`（包含非法字符）
- ❌ `5B9A-212C-54D7-ADE5-1234`（太长）

## 技术细节

### 密钥验证正则表达式

```javascript
const keyPattern = /^[A-F0-9]{4}-[A-F0-9]{4}-[A-F0-9]{4}-[A-F0-9]{4}$/;
```

- 4 组，每组 4 个字符
- 只允许 0-9 和 A-F（十六进制）
- 组之间用短横线 `-` 分隔

### 激活流程

```javascript
1. 用户输入密钥
2. 去除首尾空格，转为大写
3. 验证格式是否正确
4. 检查是否已激活其他密钥
5. 保存到本地文件
6. 更新状态为 "active"
7. 返回成功
```

### 持久化存储

许可证数据保存在：
- macOS: `~/Library/Application Support/lockscreen-focus/license.json`
- Windows: `%APPDATA%/lockscreen-focus/license.json`

数据结构：
```json
{
  "key": "5B9A-212C-54D7-ADE5",
  "instanceId": "MacBook-Pro.local",
  "firstLaunchAt": 1709097600000,
  "status": "active",
  "lastValidatedAt": 1709097600000,
  "lastValidationResult": true
}
```

## 常见问题

### Q: 激活后需要联网吗？
A: 不需要。闲鱼模式使用本地验证，激活后完全离线工作。

### Q: 密钥会过期吗？
A: 不会。本地激活后永久有效。

### Q: 一个密钥可以在多台设备上使用吗？
A: 当前版本可以。如需限制，可以添加服务器端验证。

### Q: 如何更换密钥？
A: 点击"停用许可证"按钮，然后输入新密钥激活。

### Q: 如何查看当前激活的密钥？
A: 在设置页面的"许可证信息"区域可以看到密钥的前 8 位（例如：5B9A-212...）
