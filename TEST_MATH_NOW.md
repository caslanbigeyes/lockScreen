## 快速测试（3 分钟）

### 步骤 1：启动应用

```bash
npm start
```

### 步骤 2：打开开发者工具

在应用窗口中：
- macOS: `Cmd + Option + I`
- Windows: `Ctrl + Shift + I`

或者菜单：View → Toggle Developer Tools

### 步骤 3：开始专注

1. 在"专注" Tab 中点击"开始专注"
2. 等待倒计时结束（默认 1 分钟，可以在设置中改为 10 秒快速测试）

### 步骤 4：锁屏和解锁

1. 应用会自动锁屏
2. 解锁电脑（输入系统密码）
3. 应该会弹出数学答题窗口

### 步骤 5：查看控制台日志

在**终端**（运行 npm start 的窗口）中，应该看到：

```
=== 生成数学题 ===
题目: 45 + 23
答案: 68 (类型: number)
难度: 1
================
```

### 步骤 6：输入答案

1. 在答题窗口输入正确答案（例如 68）
2. 点击"确认"或按 Enter

### 步骤 7：查看验证日志

在**终端**中，应该看到：

```
提交答案: 68 (输入值: 68)

=== 数学答题验证 ===
Payload: { answer: 68 }
用户答案: 68 (类型: number)
正确答案: 68 (类型: number)
是否通过: true
严格相等: true
==================
```

### 步骤 8：验证结果

- ✅ 如果答对：答题窗口关闭，继续专注
- ❌ 如果答错：重新锁屏

## 快速设置（方便测试）

### 缩短专注时间

1. 进入"设置" Tab
2. 将"专注时长"改为 1 分钟（或更短，最小 1 分钟）
3. 这样可以快速触发锁屏和答题

### 禁用数学答题（临时）

如果想暂时跳过答题：
1. 进入"设置" Tab
2. 关闭"数学题门禁"开关
3. 解锁后会直接继续专注，不需要答题

## 测试不同难度

数学题有 4 个难度级别：

### Level 1（试用版）
- 两位数加减法
- 例如：45 + 23 = 68

### Level 2（Pro）
- 两位数乘法
- 例如：12 × 5 = 60

### Level 3（Pro）
- 三位数加减或两位数乘法
- 例如：456 + 234 = 690

### Level 4（Pro）
- 三位数混合运算或简单平方
- 例如：12² = 144
- 例如：100 + 12 × 5 = 160

难度根据你的拖延指数自动调整（Pro 功能）。

## 常见问题

### Q: 输入答案后没反应

**检查：**
1. 是否点击了"确认"按钮或按了 Enter？
2. 查看终端是否有 "提交答案" 日志
3. 查看开发者工具控制台是否有错误

### Q: 答案明明是对的，但判错

**检查：**
1. 查看终端日志中的"正确答案"
2. 确认你输入的答案和正确答案完全一致
3. 注意负数（如果是减法）
4. 提供完整的日志信息

### Q: 看不到日志

**终端日志：**
- 主进程的日志在运行 `npm start` 的终端窗口
- 包括题目生成和答案验证

**开发者工具日志：**
- 渲染进程的日志在开发者工具的 Console 标签
- 包括输入提交信息
