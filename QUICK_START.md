# 🚀 FocusGuard 快速启动指南

## 5 分钟快速测试完整流程

### 第 1 步：启动购买页面后端（1 分钟）

```bash
# 进入购买页面目录
cd purchase-page

# 安装依赖
npm install

# 启动服务器
node server.js
```

看到以下输出表示成功：
```
购买页面后端服务运行在 http://localhost:3000
```

### 第 2 步：打开购买页面（30 秒）

在浏览器中打开：
```
purchase-page/index.html
```

你会看到一个精美的购买页面！

### 第 3 步：测试购买流程（2 分钟）

#### 方式 A：使用测试脚本（推荐）

在新终端运行：
```bash
cd purchase-page
node test-payment.js
```

你会看到完整的测试流程输出，包括生成的许可证密钥。

#### 方式 B：手动测试

1. 在购买页面点击"立即购买"
2. 会弹出支付二维码（模拟的）
3. 在新终端运行：
   ```bash
   # 使用返回的订单号（例如 FG1234567890）
   curl -X POST http://localhost:3000/api/test/pay/FG1234567890
   ```
4. 页面会自动显示许可证密钥

### 第 4 步：在应用中激活（1 分钟）

1. 复制生成的许可证密钥（格式：`XXXX-XXXX-XXXX-XXXX`）
2. 启动 FocusGuard 应用：
   ```bash
   npm start
   ```
3. 进入"设置" Tab
4. 在许可证区域粘贴密钥
5. 点击"激活"按钮
6. 看到"激活成功！"提示

### 第 5 步：验证 Pro 功能（30 秒）

激活后，你应该看到：
- ✅ 顶部横幅变为绿色："Pro 已激活"
- ✅ "随机拍照"和"解锁答题"选项可用
- ✅ 可以查看详细统计图表
- ✅ 可以导出 PDF 周报

## 🔍 常见问题

### Q1: 购买页面打开后显示"网络错误"

**原因：** 后端服务未启动

**解决：**
```bash
cd purchase-page
node server.js
```

### Q2: 点击"立即购买"没反应

**原因：** 前端 API 地址配置错误

**解决：** 打开 `purchase-page/index.html`，确认：
```javascript
const API_BASE = 'http://localhost:3000';  // 确保地址正确
```

### Q3: 许可证激活失败

**原因：** 可能是网络问题或密钥格式错误

**解决：**
1. 检查密钥格式：`XXXX-XXXX-XXXX-XXXX`
2. 确保没有多余空格
3. 打开开发者工具查看错误信息

### Q4: 如何重置测试状态

**删除许可证文件：**
```bash
# macOS
rm ~/Library/Application\ Support/lockscreen-focus/license.json

# Windows
del %APPDATA%\lockscreen-focus\license.json

# Linux
rm ~/.config/lockscreen-focus/license.json
```

## 💡 开发技巧

### 快速切换测试状态

使用测试脚本快速切换许可证状态：

```bash
# 新用户（试用期 7 天）
npm run test:license new-user

# 试用期即将到期
npm run test:license trial-6-days

# 试用期已过期
npm run test:license trial-expired

# Pro 已激活
npm run test:license pro-active
```

### 查看实时日志

```bash
# 后端日志
cd purchase-page
node server.js

# 应用日志（打开开发者工具）
# macOS: Cmd + Option + I
# Windows: Ctrl + Shift + I
```
