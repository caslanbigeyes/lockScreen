创建 `安装指南.txt` 上传到网盘：

```
FocusGuard Pro - macOS 安装指南

═══════════════════════════════════════

【系统要求】
macOS 10.15 或更高版本
支持 Intel Mac 和 Apple Silicon (M1/M2/M3)

【选择版本】
Intel Mac: FocusGuard-0.1.0.dmg
Apple Silicon: FocusGuard-0.1.0-arm64.dmg

如何查看：
1. 点击左上角  图标
2. 选择"关于本机"
3. 查看"处理器"或"芯片"
   - Intel Core → Intel 版本
   - Apple M1/M2/M3 → arm64 版本

【安装步骤】
1. 下载对应版本的 DMG 文件
2. 双击打开 DMG 文件
3. 拖拽 FocusGuard 到 Applications 文件夹
4. 从 Launchpad 或 Applications 启动

【首次运行】
macOS 会提示"无法打开，因为无法验证开发者"

解决方法：
1. 右键点击 FocusGuard
2. 选择"打开"
3. 点击"打开"按钮

或者：
1. 打开"系统偏好设置"
2. 进入"安全性与隐私"
3. 点击"仍要打开"

【激活许可证】
1. 打开应用
2. 点击顶部"设置" Tab
3. 在"许可证激活"区域输入密钥
4. 点击"激活"按钮
5. 看到"激活成功"提示

您的许可证密钥：
（见闲鱼消息）

✅ 随机拍照 - 防止假专注（Pro）
✅ 智能分析 - 拖延指数和行为模式（Pro）
✅ 详细统计 - 专注数据图表（Pro）
✅ PDF 报告 - 导出周报（Pro）

【使用方法】
1. 进入"专注" Tab

4. 解锁电脑后答题
5. 答对后继续专注

【设置说明】
- 专注时长：默认 1 分钟，可调整
- 拍照监控：Pro 功能，随机拍照
- 数学题门禁：Pro 功能，解锁答题

【常见问题】

A: 打开终端，运行：
   xattr -cr /Applications/FocusGuard.app

Q: 应用无法启动
A: 确认系统版本 >= macOS 10.15

Q: 许可证激活失败
A: 检查密钥格式（XXXX-XXXX-XXXX-XXXX）
   确保没有多余空格

Q: 如何更换设备
A: 在旧设备点击"停用许可证"
   然后在新设备激活

Q: 可以退款吗
A: 7 天内无理由退款

有任何问题请通过闲鱼联系我
我会尽快回复并帮助解决

FocusGuard Pro v0.1.0
```

**Q: 支持 Windows 吗？**

**Q: 会不会影响其他工作？**

**Q: 想换设备**

**Q: 可以退款吗**
