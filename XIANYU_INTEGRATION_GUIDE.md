### 第 4 步：生成许可证密钥

有两种方式：

#### 方式 A：手动生成（简单）

每次有人购买后，运行：

```bash
node -e "console.log(require('crypto').randomBytes(8).toString('hex').toUpperCase().match(/.{1,4}/g).join('-'))"
```

会输出类似：`A3F2-B8D1-C4E9-F7A6`

#### 方式 B：使用脚本（推荐）

创建 `generate-license.js`：

```javascript
const crypto = require('crypto');

function generateLicense() {
  const key = crypto.randomBytes(8)
    .toString('hex')
    .toUpperCase()
    .match(/.{1,4}/g)
    .join('-');
  
  console.log('\n🔑 新的许可证密钥：');
  console.log('━━━━━━━━━━━━━━━━━━━━━━━━━━━━');
  console.log(`  ${key}`);
  console.log('━━━━━━━━━━━━━━━━━━━━━━━━━━━━\n');
  console.log('📋 复制后发送给买家\n');
  
  return key;
}

// 生成 5 个备用密钥
console.log('生成 5 个许可证密钥：\n');
for (let i = 1; i <= 5; i++) {
  console.log(`${i}. ${generateLicense()}`);
}
```

运行：
```bash
node generate-license.js
```

## 🔒 防止盗版

### 1. 许可证验证

确保应用中的许可证验证功能正常：
- 每次启动验证
- 定期在线验证
- 记录激活设备

### 2. 限制激活次数

在代码中限制每个密钥只能激活 1 次：

```javascript
// 在 license.js 中
const usedLicenses = new Set();

const activate = async (licenseKey) => {
  if (usedLicenses.has(licenseKey)) {
    return { 
      success: false, 
      error: '此密钥已被使用，请联系客服' 
    };
  }
  
  // 激活成功后
  usedLicenses.add(licenseKey);
  // 保存到文件或数据库
};
```

### 3. 记录激活信息
