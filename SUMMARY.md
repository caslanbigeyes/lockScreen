## 🎯 许可证密钥生成

### 是的，支付完成后自动生成！

**生成逻辑：**
```javascript
// 在 server.js 中
function generateLicenseKey() {
  const segments = [];
  for (let i = 0; i < 4; i++) {
    segments.push(crypto.randomBytes(2).toString('hex').toUpperCase());
  }
  return segments.join('-');
}

// 生成格式：XXXX-XXXX-XXXX-XXXX
// 例如：A3F2-B8D1-C4E9-F7A6
```

**流程：**
1. 用户支付成功
2. 微信/支付宝回调后端
3. 后端验证支付
4. 自动生成唯一的许可证密钥
5. 保存到数据库
6. 返回给前端显示
7. 发送邮件给用户（可选）
